Addressing diagnostic disparities through statistically validated cross-diagnostic screening

Overview

Female autism is significantly underdiagnosed and misdiagnosed in clinical practice, with females often presenting with different behavioral phenotypes than males. Traditional diagnostic approaches fail to capture female-specific patterns, leading to missed diagnoses or misclassification as ADHD, anxiety, or mood disorders.

Designed and built statistically validated multimodal ML pipeline integrating behavioral demographics and resting-state fMRI connectivity features to identify autism biomarkers in females through A/B testing of feature modalities and connectivity networks. Applied trained model to cross-diagnostic screening of ADHD population to flag potential misdiagnosed cases.

Multimodal Feature Set

  • Age at scan
  • Full-scale IQ, Verbal IQ, Performance IQ

  • DMN Within-Network
  • Salience Network
  • Frontoparietal Network
  • Inter-Network Connectivity
  • Edge Density Metrics
  • Global Connectivity

A/B Testing Framework

Test 1 - Feature Modality:

  • Condition A: Demographics only (4 features)
  • Condition B: Multimodal (demographics + connectivity, 23 features)
  • Statistical validation: Paired t-test + Wilcoxon signed-rank test
  • Result: No significant difference (p=0.60), validated multimodal selection

Test 2 - Network Scope:

  • Condition A: DMN features only
  • Condition B: Three networks (DMN + SN + FPN)
  • Statistical validation: Paired t-test + Wilcoxon signed-rank test
  • Result: No significant difference (p=0.63), validated three-network approach

Implementation

Data Preprocessing

Implemented robust data cleaning pipelines for ABIDE (Autism Brain Imaging Data Exchange) and ADHD-200 datasets. Handled missing values through median imputation for demographics and careful NaN cleaning for neuroimaging data. Extracted ROI timeseries from resting-state fMRI across multiple sites, harmonizing data from different scanner protocols and acquisition parameters.

Feature Engineering

Designed atlas-agnostic network-level connectivity features enabling seamless application across different neuroimaging parcellations. Extracted within-network connectivity metrics (mean, std, max) for Default Mode Network (DMN), Salience Network (SN), and Frontoparietal Network (FPN). Computed inter-network connectivity patterns and graph theory metrics including edge density and node degree distributions.

Statistical Validation

Implemented comprehensive A/B testing framework with paired statistical tests (t-test + Wilcoxon signed-rank) to systematically compare feature modalities and network configurations. Validated biomarker candidates using Mann-Whitney U tests (p < 0.05) with effect size calculations (Cohen's d). Prioritized scientific rigor over performance optimization, ensuring reproducible and clinically meaningful results.

Model Training & Validation

Trained XGBoost classifier selected for interpretable feature importance metrics and built-in regularization to prevent overfitting on small sample (N=157). Employed stratified 5-fold cross-validation to ensure robust performance estimation across different data partitions. Extracted feature importance rankings and validated top biomarkers through independent statistical tests.

Cross-Diagnostic Application

Applied trained model to ADHD-200 dataset, extracting identical 23-feature set from independent neuroimaging data processed with different atlas (ROI 1000 vs CC200). Generated autism probability scores for each subject, ranking by likelihood and flagging top 20% as high-risk cases exhibiting autism-signature patterns warranting clinical review.

Key Technical Decisions

01 — Atlas-Agnostic Network Features

Designed network-level connectivity features (DMN, SN, FPN) that generalize across different brain parcellations, enabling successful transfer from ABIDE's CC200 atlas (200 ROIs) to ADHD-200's ROI 1000 atlas (1000 ROIs) without model retraining. This architectural choice prioritized cross-dataset applicability over atlas-specific optimization.

02 — Biomarker-Based Approach

Deliberately excluded diagnostic instruments (ADOS, ADI-R) from feature set despite their high predictive power to avoid data leakage and enable true biomarker discovery. This decision prioritized scientific validity and cross-diagnostic applicability over performance metrics, as diagnostic instruments are unavailable in non-autism populations.

03 — Statistical Rigor Over Performance

Implemented comprehensive statistical validation (A/B testing, cross-validation, biomarker validation) rather than aggressive hyperparameter tuning. Accepted realistic performance metrics while ensuring reproducible methodology and honest reporting of results, contributing to more reliable medical machine learning practices.

Successfully developed statistically validated multimodal ML pipeline through rigorous A/B testing framework. Validated pipeline through 5-fold cross-validation, demonstrating reproducible results across training folds while maintaining scientific rigor and avoiding overfitting.

Successfully applied pipeline to cross-diagnostic screening of ADHD-200 dataset, identifying 20% of subjects with autism-signature patterns warranting clinical review. Top-scoring cases exhibited autism probabilities of 94%, 92%, and 77%, demonstrating pipeline's capability for misdiagnosis identification in underrepresented populations.

Biomarker validation identified verbal IQ as statistically significant discriminator (p=0.03, Cohen's d=-0.37). A/B testing revealed no significant differences between modalities (p=0.60) or network configurations (p=0.63), validating selection of multimodal three-network approach based on neuroscientific rationale rather than performance optimization alone.

Clinical Utility

Pipeline generates interpretable autism probability scores suitable for clinical review, providing objective data to supplement diagnostic assessment. Identified 12 high-risk ADHD cases with concrete probabilities, enabling clinicians to prioritize subjects for comprehensive autism evaluation and potentially identify misdiagnosed individuals.

Methodological Contributions

Demonstrated feasibility of cross-diagnostic screening using autism-trained models on independent neurodevelopmental populations. Established reproducible A/B testing framework for feature selection validation. Contributed to honest reporting practices in medical ML by prioritizing scientific validity over inflated performance metrics.

Impact

Developed data-driven approach to identify potential misdiagnosed cases in underrepresented female populations where traditional diagnostic pathways often fail. Pipeline provides objective, reproducible screening tool to supplement clinical assessment and address diagnostic disparities.

Demonstrated cross-diagnostic screening framework applicable to other neurodevelopmental populations, establishing methodology for identifying shared phenotypic patterns across diagnostic categories. This approach enables systematic flagging of cases for clinical re-evaluation and potential diagnostic revision.

Implemented comprehensive statistical validation framework establishing best practices for medical ML research. Prioritized reproducibility and scientific rigor over performance metrics, contributing to more reliable and clinically meaningful machine learning applications in healthcare.

Future Directions

  • Scale to larger multi-site datasets (ABCD, HBN) with thousands of subjects to improve model robustness and generalizability
  • Expand to mood disorder populations (depression, anxiety) pending access to datasets with comprehensive phenotypic and neuroimaging data
  • Incorporate task-based fMRI features (social cognition, emotion processing) to complement resting-state connectivity patterns
  • Conduct prospective clinical validation studies comparing flagged cases against expert diagnoses to measure real-world screening accuracy
  • Develop explainability tools for clinical integration, enabling interpretable feature contributions at individual subject level
