NLP Pipeline for Employee Survey Intelligence

Problem

HR teams conducting employee pulse surveys faced a qualitative analysis bottleneck: thousands of open-ended responses about career development, engagement, and organizational concerns required manual categorization, delaying insight delivery and limiting scalability. Traditional manual coding approaches couldn't handle the volume of rich qualitative feedback, introduced subjective bias, and prevented rapid turnaround from survey close to actionable insights.

Solution

Designed and deployed a two-phase NLP pipeline combining unsupervised topic discovery with ensemble-based supervised classification to automate qualitative feedback analysis at scale. The system processes employee survey responses through automated theme extraction followed by weighted ensemble classification to assign the top 3 most relevant topics to each comment.

Two-Phase Architecture

PHASE 1

Unsupervised Topic Discovery

Automated theme extraction from employee feedback using dual topic modeling approaches

Models: LDA (scikit-learn) + BERTopic (sentence transformers)
Preprocessing: NLTK with 4 configurable levels
Output: Discovered topic themes with coherence metrics

PHASE 2

Supervised Topic Classification

Ensemble-based classification assigning top 3 topics per response using weighted model predictions

Models: 3 sentence transformers (71%, 71%, 69% weights)
Method: Cosine similarity-based classification
Output: Top 3 topics per comment with confidence scores

Two-Phase NLP Pipeline Architecture

Implementation

Phase 1: Unsupervised Topic Discovery

Built modular NLP pipeline with configurable preprocessing levels (minimal, basic, standard, hr_focused) using NLTK. Implemented dual topic modeling approach combining LDA (scikit-learn) and BERTopic (sentence transformers) to extract themes from employee feedback. LDA provided interpretable linear topic models while BERTopic leveraged transformer-based embeddings for semantic topic discovery. Pipeline automatically generated coherence metrics to evaluate topic quality and optimize the number of discovered themes.

Phase 2: Supervised Ensemble Classification

Designed 3-model ensemble combining sentence transformer models with performance-based weighting. Each model computed cosine similarity between comment embeddings and topic embeddings, with final predictions aggregating weighted similarity scores across all three models. System assigned top 3 most relevant topics to each comment with confidence scores, enabling granular analysis of multi-faceted employee feedback.

Ensemble Models:

  • nli-mpnet-base-v2 (71% weight): Natural language inference model optimized for semantic similarity
  • stsb-mpnet-base-v2 (71% weight): Semantic textual similarity model trained on sentence pairs
  • nli-roberta-large (69% weight): Large-scale RoBERTa model for robust representations

Modular Architecture

Implemented reusable pipeline components for maximum flexibility: data_loader (Excel/CSV input handling), text_processor (NLTK preprocessing), topic_modeler (LDA + BERTopic orchestration), results_handler (Excel multi-sheet output). Deployed on Dataiku backend for managed folder integration and scheduled execution.
